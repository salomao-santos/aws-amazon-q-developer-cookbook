# Testing Recipes

This directory contains tested prompts for generating comprehensive tests with Amazon Q Developer.

## Available Recipes

1. [Unit Testing](./unit-testing.md) - Create unit tests for individual functions/classes
2. [Integration Testing](./integration-testing.md) - Test component interactions
3. [API Testing](./api-testing.md) - Test REST and GraphQL APIs
4. [E2E Testing](./e2e-testing.md) - End-to-end testing with Playwright/Cypress
5. [Test Data Generation](./test-data.md) - Generate realistic test data

## Usage

Each recipe includes:
- **Prompt Template**: Ready-to-use prompt for Amazon Q Developer
- **Test Examples**: Sample test cases
- **Best Practices**: Tips for effective testing
- **Coverage Guidelines**: How to achieve comprehensive coverage
