# Code Creation Recipes

This directory contains tested prompts for creating high-quality code with Amazon Q Developer.

## Available Recipes

1. [API Endpoint Creation](./api-endpoint.md) - Create RESTful API endpoints with proper error handling
2. [Database Schema Design](./database-schema.md) - Design and implement database schemas
3. [Lambda Function Creation](./lambda-function.md) - Create AWS Lambda functions with best practices
4. [React Component Creation](./react-component.md) - Build modern React components
5. [Python Class Creation](./python-class.md) - Create well-structured Python classes

## Usage

Each recipe includes:
- **Prompt Template**: Ready-to-use prompt for Amazon Q Developer
- **Expected Output**: What you should receive
- **Best Practices**: Tips for optimal results
- **Examples**: Real-world use cases
